GEP SISTEMA ONLINE

1. Suba esta pasta em um repositório no GitHub.
2. No Netlify, use Import from Git.
3. Confirme a variável DATABASE_URL no Netlify.
4. Publique.

Arquivos principais:
- index.html
- style.css
- script.js
- package.json
- netlify.toml
- netlify/functions/state-get.js
- netlify/functions/state-save.js

Observação:
O arrastar-e-soltar do Netlify normalmente não instala a dependência pg. Para esta versão, prefira GitHub + Netlify.
